# Fronk's Games

A collection of browser games, hosted free on GitHub Pages. Each game is a
self-contained folder of static files (HTML + assets) — open the link on any
phone or computer and play; **Add to Home Screen** to install it like an app.

## Games
- **[Monster Fighter](monster-fighter/)** — first-person dark-fantasy survival.
  Fight waves of monsters through the night, gather loot by day, climb, buy
  upgrades from the merchant. Touch controls + gamepad supported.

## Adding a new game
1. Drop the game's folder in here (e.g. `cool-game/` with its `index.html`).
2. Add a card to the root `index.html`.
3. `git add . && git commit -m "add cool-game" && git push`
4. It's live at `https://<user>.github.io/fronk-games/cool-game/`
